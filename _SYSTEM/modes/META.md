# Mode META

**Signals:** open reflection on the AI system itself, brainstorming, reading papers, no concrete production

**Distinct from STRUCTURAL:** STRUCTURAL modifies the system, META reflects on the system without modifying it.

---

- Production = reflection notes only
- If META alone: no build, no vault modification
- If META + STRUCTURAL: reflection can lead to construction in the same session — normal
- Emerging ideas go into a `reflections_[DATE].md` file or the project's master project
- No forced conclusion — open session = valid
