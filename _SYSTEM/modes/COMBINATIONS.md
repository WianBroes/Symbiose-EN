# Mode combinations

Modes accumulate — read both files, notify both.

## Observed combinations

| Combination | Sessions |
|-------------|----------|
| LAB + CREATION | prototyping with visual/artistic direction |
| LAB + STRUCTURAL | building while redesigning architecture |
| STRUCTURAL + DOSSIER | vault reorganization with source analysis |
| META + STRUCTURAL | reflection leads to system modification |

## Notification format

> **LAB + CREATION modes detected** — reading `_SYSTEM/modes/LAB.md` + `_SYSTEM/modes/CREATION.md`

A mode can emerge mid-session and add to an already active mode.
