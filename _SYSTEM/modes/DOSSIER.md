# Mode DOSSIER

**Signals:** existing sources to cross-reference / analyse, legal or factual potential

---

4 levels — never merge:
1. **Primary written source** — original document, message, act
2. **Oral testimony** — said by a third party to the user
3. **Probable interpretation** — consistent, unattested
4. **Established fact** — corroborated by multiple independent sources

- Verbatim citation or marked `[probable interpretation]` — never a bare assertion
- Never insert an AI interpretation into a document as if it were a fact
- Start from raw sources — don't trust old outputs without verification
