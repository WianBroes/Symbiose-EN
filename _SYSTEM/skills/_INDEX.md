# Skills — Index

Skills are procedure modules loaded on demand. Each folder contains a `SKILL.md` with instructions.

## Available

| Skill | File | Description |
|-------|------|-------------|
| **import** | `import/SKILL.md` | Document import and indexing. Converts PDF, DOCX → markdown, preserves originals. |
| **export** | `export/SKILL.md` | Profile, framework export, or backup. Prepares migration or publication. |
| **closure** | `closure/SKILL.md` | Session closure ritual. Macro-scan, traits, skills, TRANSFERT, snapshot. Triggered by "close", "we're done". |
| **dream** | `dream/SKILL.md` | Observation consolidation (append → compact). Triggered automatically every 10 closures, or manually by "dream". |

## Usage

The AI loads a skill when context demands it. No automatic loading — progressive disclosure: only the description is visible, content is read on demand.
